# Module 1: Entity-Relationship Model and Relational Schemas

## Learning Outcomes

By the end of this lesson, you will be able to:

- Understand the basics of the Entity-Relationship Model
- Understand relationship cardinalities and how they define how data is stored in a database
- Understand how tables are structured in a relational database
- Understand the concept of primary and foreign keys and how they are used to implement relationships in a database
